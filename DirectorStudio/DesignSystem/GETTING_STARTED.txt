🎬 WHAT IS LENSDEPTH?

A cinematic design system that makes the UI recede, putting your film at 
the forefront. Built on psychological principles and professional 
cinematography aesthetics.

📚 START HERE:

1. README.md - Overview and quick start
2. LensDepthSystem.md - Full design specification  
3. QUICK_REFERENCE.md - Daily developer cheat sheet
4. Documentation/IMPLEMENTATION_GUIDE.md - How to build
5. Documentation/EXAMPLES.md - Real-world code

🎨 CORE COLORS:

Dark Mode (Default):
  • Background: #191919 (near-black, reduces eye strain)
  • Surface: #262626 (panels and modals)
  • Primary: #FF9E0A (warm amber - focus and concentration)
  • Secondary: #4A8FE8 (cool blue - trust and logic)
  • Success: #32C48D
  • Danger: #C74432
  • Text Primary: #F0F0F0
  • Text Secondary: #A0A0A0

📏 SPACING (8px Grid):

  • Unit: 8px
  • Inner: 16px (component padding)
  • Outer: 24px (between modules)
  • Margin: 32px (section margins)

🧩 QUICK USAGE:

Swift:
  LensDepthTokens.colorPrimaryAmber
  LensDepthTokens.spacingOuter
  
  LDPrimaryButton(title: "Generate", action: { })
  LDPanel { /* content */ }

✅ CHECKLIST FOR NEW COMPONENTS:

  □ Uses design tokens (no hardcoded values)
  □ Follows 8px spacing grid
  □ Includes visual depth (shadows)
  □ Meets WCAG contrast (7:1 for text)
  □ Has accessibility labels
  □ Works in light/dark mode
